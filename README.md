# 🎓 Sheryians Coding School — Landing Page Clone

A responsive landing page I built for fun, replicating the [Sheryians Coding School](https://sheryians.com) website using plain **HTML** and **CSS**.

---

## 🚀 What I Built

A pixel-close clone of the Sheryians Coding School homepage featuring:

- Fixed navigation bar with logo and links
- Hero section with course info, tags, pricing, and CTA buttons
- Course detail card (language, certification, schedule, content count)
- Fully responsive layout for mobile and desktop

---

## 🛠️ Tech Stack

| Tech | Usage |
|------|-------|
| HTML5 | Page structure |
| CSS3 | Styling, layout, responsiveness |

No frameworks. No libraries. Just vanilla HTML & CSS. 💪

---

## 📱 Responsive Design

- **Desktop**: Two-column hero layout with `flexbox` and `space-between`
- **Mobile** (`max-width: 768px`): Navbar collapses to icon-only view, layout stacks vertically

---

## 💡 Things I Learned / Practiced

- Flexbox layouts and alignment
- Fixed navbar with proper spacing using `padding-top` on `main`
- Responsive design with `@media` queries
- Why `justify-content: space-between` needs **both children to have explicit widths** to work correctly
- Using `object-fit: cover` and `display: block` on images to remove unwanted gaps
